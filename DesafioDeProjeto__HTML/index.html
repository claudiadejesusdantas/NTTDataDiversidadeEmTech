Estrutura básica do HTML

Uma convensão na hora de criar um arquivo HTML para trabalhar é chamá-lo de index.html. Toda tag html tem um ponto inicial e um ponto final, mostrado por uma barra /. O html consiste em duas partes cabeça (head) e corpo (body). Tudo que estiver no corpo é o que de fato aparece para o usuário e na cabeça fica as informações de metadados, o único dado do head que aparece para o usuário é o title que é o título da página na barra do navegador.

Falando sobre tags e atributos

  • HTML tudo que estiver escrito aqui dentro será uma página html.
  • HEAD local onde fica os metadados da página.
  • BODY corpo do site, parte vísivel para o usuário.
  • Tag i significa itálico.
  • Tag strong significa negrito e forte.
  • Tag u significa sublinhado.
  • Tag input é para colocar informações, campo de texto e não tem fechamento.
  • Tag sup é para colocar textos subscritos, renderizado em uma fonte menor, usados para notas de rodapé.
  • Atributo type é para colocar o tipo de informação que será colocado no input, o mais usado no curso será text e number.
  • Tag img é para colocar imagem, ela precisa de um src que é o endereço da imagem e um alt que é a descrição da imagem,essa tag não tem fechamento.
  • Tag id é para colocar como um "rg" no seu elemento, um identificador único.
  • Tag style é para colocar estilização(CSS) no seu html de forma direta no código.
  • Tag class é para colocar estilização(CSS) de uma maneira padronizada em mais de um elemento no seu html.
  • Tag color é para colocar cor no elemento junto com o style.

Textos

H! ...H6 são os títulos da página, sendo o H1 único, sendo o mais importante da página e os outros subtítulos.
p uma tag usada para escrever parágrafos.
responsividade uma forma de utilizar o site em vários telas como celular ou tablet.
blockquote para referenciar alguma parte do texto, utilizado em citações.

Links

A tag a em html é de anchor que significa um hiperlink que é usado para ir para o endereço registrado no href, tendo a opção de ser aberto na mesma página ou em outra guia com o elemento target.
